# Proposito del proyecto

Un juego de aventura con IA. Inspirado en el juego clasico de texto Zork, vitaminado con IA. Cada partida debe ser una experiencia nueva. El entorno debe ser variado, una mazmorra medieval, una nave espacial abandonada, una ciudad subterranea, etc. El juego debe ser jugable en la consola de comandos.

Se debe primar los quest lógicos a la tirada de dados.
